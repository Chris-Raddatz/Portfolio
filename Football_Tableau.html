European Football Analysis

This short project was completed using SQL queries to clean and aggregate a SQL database to then create a Tableau dashboard visualizing insights across European football.

The quick summer project was combination of SQL and Tableau. As an avid soccer fan I wanted to create something relevant to the sport. I found this Kaggle database and used SQL to explore the data and aggregate it into more easily digestable tables. I exported the data to Tableau and created a dashboard looking into the top European leagues, physical characteristics of elite players and top teams within each of the leagues. I do have a small Github repo for this project linked here. I also wrote a Medium article on the project. The image above is one of the sheets from the dashboard I built.
