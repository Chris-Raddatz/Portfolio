Politician Portfolio Analysis

Completed with 3 other students, we investigated politicians spending data in R via data from CapitolTrades.

A project for the 6203 course in OMSA - Business Analytics. We investigated politician's spending behavior by creating porfolios based off their trading activity and comparing it to the S&P 500. We constructed Beta values for groups of politicians by party and age, investigating how risky or even profitable these groups were compared to the general stock market. All analysis was done in R, with some visuals created in Tableau. The code is available on this Github Link. I also personally created a Power BI dashboard with this data, looking into activity by party, gender and how that affects investing behavior. I wrote an article on Medium regarding the dashboard.
