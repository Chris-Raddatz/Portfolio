NFL Big Data Bowl Analysis.

An attempted project for the NFL Big Data Bowl, I used Python to analyze football players, identifying possible rising stars within the last few seasons.

A short personal project that I did without submitting an actual story to the NFL Big Data Bowl. I used the data provided to build some visuals with packages like Matplotlib and Seaborn, then used KMeans clustering to identify clusters of players. I clustered corner backs and located the cluster with a majority of players being in the PFF Top Corners list. Within those clusters I looked at players that weren't recognizable but had great seasons, and validated whether or not they were later recognized as top players. With this approach, one could use clustering to search for hidden stars across the NFL. The code is available via Jupyter Notebook here. I also wrote a short Medium article on the project.
