Housing Prices Analysis

A course project where I investigated Housing Prices and their possible correlation with inflation. Analysis performed with mostly Python and R.

This project was part of my course in OMSA ISYE 6740 - Computational Data Analytics. I used Python to combine two separate dataframes and analyze them. I first used Pandas for data cleaning and combination, then Matplotlib and Seaborn to create the visuals. I then used a couple machine learning techniques, such as Principle Component Analysis (PCA), KMeans clustering, Kernel Density Estimation and Linear Regression to conduct statistical analysis on the data. My favorite technique out of those was KMeans, as it generated clusters of states to identify similar housing markets across the country. You can also see the code, data and visuals I produced on this Github Link.
