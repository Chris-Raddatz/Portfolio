SQL Database Creation and Website Development

Another course project in OMSA, completed with 3 other students. We created a SQL database from concept to application. Database creation was handled with postgreSQL. We then created a local bidding website with Javascript, Node.JS and HTML.

A project for the 6400 Database Systems Concept and Design course in OMSA, I did this project with 3 other students in Spring 2024. We were tasked with creating a database from scratch, from IFD and EER diagrams all the way to SQL queries and connections. The website was called "BuzzBid", a bidding website for users to sell and buy items from each other. We had to make sure that our database could handle inserts, deletions and special conditions according to the course instructions. Database connections were handled with postgreSQL and web development connected to said database with Node.JS and Javascript, along with HTML for website appearance. Version control was handled with Git. The code is available on this Github Link.
